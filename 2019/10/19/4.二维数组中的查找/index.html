4.二维数组中的查找

发表于 2019-10-19 更新于 2019-10-29 分类于 剑指offer 阅读次数：
本文字数： 1.2k 阅读时长 ≈ 1 分钟

题目描述
在一个二维数组中（每个一维数组的长度相同），每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。
Python
遍历二维数组

规律：首先选取数组右上角的数字

  • 如果该数字等于要查找的数字，则查找结束
  • 该数字大于要查找的数字，则剔除这个数字所在的列
  • 该数字小于要查找的数字，则剔除这个数字所在的行 C++
